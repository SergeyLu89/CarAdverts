# Car advert app

### Welcome to our car rental application

![home page screen](/public/readme/home_page.jpg)

This application allows you to quickly select a car for rent from the proposed options.

## Functional:
![catalog page screen](/public/readme/catalog_page.jpg)
The "catalog" page displays announcement cards.
Each card can be added to favorites by clicking on the button with the image of a heart.
You can also find out more detailed information about the card by clicking on the “Learn more” button.
Scroll to the end of the page, you can download additional cards by clicking on the "Load more" button.

![favorite page screen](/public/readme/favorite_page.jpg)
The "Favorites" page displays cards added to favorites.
